# 💐 Flores con Amor

Este proyecto es una pequeña web interactiva que permite **enviar flores con amor y cariño**, totalmente hechas con **HTML y CSS**, sin necesidad de imágenes externas. Cada flor es creada con estilos CSS puros y animaciones suaves para dar vida a un gesto bonito y virtual.

---

## 🌷 Características

- Flores hechas **100% con CSS**.
- Animaciones suaves para dar **vida y movimiento**.
- Interfaz sencilla e intuitiva para enviar flores virtuales.
- Perfecto para **regalos digitales, mensajes de amor o amistad**.
- Compatible con los navegadores modernos.

---

## 🚀 Cómo usar

1. Ingresa a la web:
   https://hafidod.github.io/flower-app/inicio.html

2. Escoge el motivo

3. Escoge una frase o escribe una personalizada

4. Ingresa el nombre a quien le quieres enviar el detalle

5. Ingresa tu nombre, esto es opcional

6. Comparte el link con la persona especial
